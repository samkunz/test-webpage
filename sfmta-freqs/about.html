San Francisco Transit Frequencies

To create this demo, I used Javascript, HTML, CSS, Python, Leaflet, geojson, and Carto base maps. More specifically, I used:

  • The Leaflet library for a slippy map.
  • The Carto Positron base map.
  • HTML and CSS for the webpage.
  • Javascript and Leaflet to dynamically add/remove lines depending on value of range (slider) and checkbox inputs.
  • Javascript and Leaflet to dynamically add popup and line highlighting when a given route is selected.
  • Python to process San Francisco transit data from multiple sources (geospatial, frequency) to output as geojson for adding Leaflet layers.

Goals & Non-Goals:

  • Goal: Show my interest in cartography, public transit, and interactive data visualization.
  • Goal: Demonstrate familiarity with web dev tech (e.g. HTML, CSS, Javascript) and cartography tech (e.g. Leaflet, geojson).
  • Non-Goal: Include all the complexity and edge cases of the San Francisco transit network.
  • Non-Goal: Create a webpage that looks great on mobile.
